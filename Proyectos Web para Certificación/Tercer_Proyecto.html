Introducción

El mundo del desarrollo web es un mundo genial en donde se puede explorar muchas cosas, se puede llegar a adquirir una logica de programación en el cual te ayudara mucho en un futuro, como herramienta tecnologica JavaScript es un lenguaje de programación que te brindara muchas herramientas para el desarrollo web, ¿Que estas esperando? Ven y adentrate a este mundo de la programación frontend.

¿Que es JavaScript?

JavaScript es un lenguaje de programacion orientado a objetos se usa en la programación de paginas web, creación de aplicaciones web, creación de video juegos y para la interactividad en las paginas web

Tipos de datos

En JavaScript existen tipos de datos como en otros lenguajes aunque la manera de declararlos varian o puede decirce que es mas sencilla aqui te mostramos cuales son esos tipos de datos.

  • Number: Es un tipo de dato numerico ya sea entero o real.

  • String: Es un campo de caracter o cadena de caracteres

  • NULL: El valor de null representa la ausencia de valor

  • Undefine: Esto ocurre cuando no tiene un valor asignado y es muy distinto que NULL

  • NAN: Esta son las siglas en ingles de (Not A Number) y significa en español que no es un número.

  • Booleano: Es un valor logico de True (verdadero) o ya sea False (Falso)
Pero todo esto se puede usar ya sea con Var, Let y Const las dos primeras son para declarar variables y la tercera es para declarar una constante, eso es un dato que no cambia.

Scope

Al hablar sobre el Scope nos referimos al alcance que puede tener una variable, hay dos tipos de Scope:

Scope local: Es cuando una variable es local a un modulo o una funcion y no se usa ese valor si no solo en el modulo declarado.
Scope global: Tiene mayor alcance ya que como dice la palabra es global y su valor se puede ocupar en cualquier modulo, no tiene limitaciones.

  4. / División
  5. % Residúo

Operadores De Relación

  2. <= (Menor o igual que)

  5. == (Igual)

Operadores Lógicos

Selección

La Selección se aplica cuando hay uno, dos o mas caminos que se pueden tomar o mejor dicho desiciones por tomar ellas son conocidas como condicionales ya que con condicionan una variable o un dato y lo convierten en booleano si es true significa que es verdadero y tiende a entrar en la condición es caso de que ella se cumpla, en caso contrario buscara otro camino hasta que el se cumpla.
Tambien hay otra tecnica que solo se utiliza en categorias ella es la tecnica de programación Switch y aqui estan ejemplo de cada una de las tecnicas:

        
            Selección Simple
            
            var number=8;

            if(number<=5)
            alert('\nNumero menor que 5\n');
        
    
        
            Selección Doble

            let number=8;

            if(number<=7)
            alert('\nNumero menor que 5\n');
            else
            alert("Numero igual o mayor que 8");
        
    
        
            Selección Múltiple

            const num=8;

            if(num<=7)
                alert('\nNumero menor que 5\n');
            else
                if(num>=8)
                    alert("Numero igual o mayor que 8");
                else
                alert("El dato no es un numero");

Repetición

La técnica de Repetición se usa al momento de realizar una tarea mas de una vez ellos son conocidos como bucles,ciclos,loop,iteración y lazo estas estructuras repetitivas son:

  • Ciclo for
  • Ciclo while
  • Ciclo do while
Tambien existen bloques de asignación especiales como es el contador (cont) que cuenta el numero de vecez que se ejecuta una tarea y el acumulador (acum) que lo que hace es acumular datos ya sean reales o enteros pero no caracteres.

Tanto el while como el do while pueden ser controlados por un contador, centinela o ya sea por un acumulador. La estructura basica de cada uno de ellos es la siguiente. 
            
                Bucle for

                var i;

                for(i=1;i<=5;i++)
                {
                    //Conjunto de acciones por hacer
                }
            
        

            
                While controlado por un contador

                let cont=1;

                while(cont < 10)
                {
                    //Conjunto de acciones a realizar
                    cont+=1;
                }
            
        

            
                While controlado por un centinela

                var centi=prompt("Ingrese un numero distinto a 0");

                while(centi!=0)
                {
                    //Conjunto de acciones a realizar
                    var centi=prompt("Ingrese un numero distinto a 0");
                }
            
        

            
                While controlado por un acumulador

                let suma=0;

                while(suma < 100)
                {
                    var valor=prompt("Ingrese un valor");
                    suma+=valor;
                }
            
        

            
                Do while controlado por un centinela

                do{
                    //Conjunto de acciones a realizar
                    var centi=prompt("Ingrese un numero distinto a 0");
                }
                while(centi!=0);

Estas tecnicas son utiles al momento de realizar una tarea muchas veces al igual se pueden anidar bucles.

Funciones

Una función es aquella la cual un problema se vuelve en subproblemas o mejor dicho en problemas mas pequeños tambien es conocida por el nombre de "Divide y Venceras". Una función puede ser llamada o invocada por medio de un bloque de asignación o por medio de una instrucción de salida.

Parametros

En las funciones exiten el paso de parametros y en JavaScript son parametros por valor que se le mandan a las funciones pero en otros lenguajes de programación como C o C++ existe el paso de parametros por referencia lo lindo de este lenguaje de programación como es JavaScript es que, es dínamico e incluso con una estructura de datos llamada arreglo se puede definir un tamaño o no, ya que el es dínamico. Estas son algunas de las caracteristicas del paso de parametros:

  • Mismo tipo de datos tanto en la llamada de la función como en la definición de ella.
  • Mismo orden en el paso de parametros.
  • Misma cantidad de parametros tanto en la definición del modulo. como en la llamada a la función.

Uso de function

En JavaScript existe una palabra reservada que es function ella lo que hace es que indica que es una función este es un breve ejemplo de una función que sumara dos numeros: 

            
                function sumar (n1,n2)
                {
                    return n1+n2;
                }

                var recibe=sumar(5,9);
                console.log("El resultado de la suma es: %d",recibe)
Ten en cuenta esto, esta función solamente esta sumando dos números pero el enfoque de ellas es resolver problemas mas complejos con con este breve ejemplo sencillo te puedes dar cuenta de como trabaja, ademas la función tiene un especificador de formato y esto no esta malo ya que se puede hacer y es mejor a si especificamos que valor mandara por pantalla esto lo hereda de su abuelo C. Lo que es el punto y coma (;) JavaScript es flexible con esto ya que no importara si se coloca o no el lo coloca al final, pero por buena practica se hace.

Este es un ejemplo con la misma función pero tecleado por el usuario

            
                function sumar (n1,n2)
                {

                    return parseInt(n1) + parseInt(n2);
                    /*Esto es para que lo lea como un numero y no como una letra ya que 
                    el signo '+' significa concatenación tambien.*/
                }

                var num1=prompt("\n\tTeclee un número:");
                var num2=prompt("\n\tTeclee otro número:");
                var recibe =sumar(num1,num2);
                console.log("El resultado de la suma es: %d",recibe);
prompt es una palabra reservada que ya la hemos ocupado en otros ejemplos lo que hace es que le pide al usuario te digite algo, ya sea un numero entero o real y tambien permite ingresar un texto o mejor dicho una cadena de caracter.

Función de flecha

Esto es lo mismo que una función común solo que la sintaxis cambia un poco, este es un ejemplo con las misma funciones de antes:

        let sumar= (n1,n2)=>
            {
                return n1*n2;
            }

            var recibe=sumar(5,9);
            console.log("El resultado de la suma es: %d",recibe)
        

        let sumar = (n1,n2) =>{

                return parseInt(n1) % parseInt(n2);//Este es el módulo
            }

            var num1=prompt("\n\tTeclee un número:");
            var num2=prompt("\n\tTeclee otro número:");
            var recibe =sumar(num1,num2);

Objetos

Los objetos en JavaScript son colecciones de propiedades, un objetos esta compuesto por su nombre, una clave y un valor que posee esa clave.

Ejemplo de un objeto

            const persona = /*Es mi objeto*/
            {
                /*Es mi clave -> */ edad=20, /*Es el valor de mi clave*/
                sexo:"Masculino",
                nombre:"Jose",
                altura:1.7
            }

            Podemos acceder a un objeto de dos maneras
            
            accediendo a todo el objeto.
            
            console.log(persona);

            accediendo una parte del objeto 1ra forma 
            
            console.log(persona.nombre);

            accediendo a una parte del objeto 2da forma
            
            console.log(persona["sexo"]);
